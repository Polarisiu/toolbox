# 甲骨文 VPS 一键管理脚本

用于 Oracle Cloud (甲骨文) VPS 的一键管理脚本，集成了多种常用功能：救砖、系统重装、保活、抢机安装等。  
通过简单的菜单选择，即可完成对应操作。

---
---

## 🚀 使用方法

### 一键脚本
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/sistarry/toolbox/main/Oracle/oracle.sh)
```


## 使用说明

### 快捷命令

  - `o`
  - `O`

安装完成后，只需在终端输入 `o` 或 `O` 即可快速启动主菜单。
